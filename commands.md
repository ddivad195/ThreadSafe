# Commands

## Key 
| Symbol      | Meaning                        |
|-------------|--------------------------------|
| [Argument]  | Argument is not required.      |

## Configuration
| Commands | Arguments                   | Description                      |
|----------|-----------------------------|----------------------------------|
| setup    | user-channel, staff-channel | Set up a guild to use ThreadSafe |

## Thread
| Commands | Arguments | Description                     |
|----------|-----------|---------------------------------|
| stats    | [Channel] | View thread stats for a channel |
| thread   | Message   | Post message as a thread        |

## Utility
| Commands | Arguments | Description             |
|----------|-----------|-------------------------|
| Help     | [Command] | Display a help menu.    |
| info     |           | Bot info for Threadsafe |
